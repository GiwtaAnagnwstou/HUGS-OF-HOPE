# Hugs of Hope – Animal Shelter Website

Αυτό το project είναι μια εκπαιδευτική ιστοσελίδα του προγράμματος "Web Developer", για το καταφύγιο ζώων "Αγκαλιά Ελπίδας".  
Στόχος είναι η παρουσίαση του έργου του καταφυγίου, η προώθηση υιοθεσιών και η ενημέρωση για δράσεις και εκδηλώσεις.

---

## Live Website
👉 https://giwtaanagnwstou.github.io/HUGS-OF-HOPE/

---

## Σελίδες

- 🏠 **Home** – βασική παρουσίαση καταφυγίου  
- ℹ️ **About Us** – πληροφορίες για το έργο  
- 📞 **Contact** – φόρμα επικοινωνίας & χάρτης  
- 💸 **Donate** – σελίδα δωρεών  
- 📅 **Events** – εκδηλώσεις και δράσεις  

---

## Τεχνολογίες

- HTML5  
- CSS3  
- Bootstrap  
- GitHub Pages (hosting)

---

## ⚠️ Σημείωση

Η ιστοσελίδα δημιουργήθηκε για **εκπαιδευτικούς σκοπούς** και δεν υποστηρίζει πραγματικές συναλλαγές ή backend λειτουργίες.

---

## Μελλοντικές βελτιώσεις

- Backend για φόρμα επικοινωνίας  
- Πραγματικό σύστημα δωρεών  
- Responsive βελτιώσεις  
- Δυναμικό περιεχόμενο
